# Инструкция по редактированию сайта

## Быстрый доступ к редактированию:
1. Откройте файл на GitHub: https://github.com/U007U/Syte/edit/main/assets/js/data.js
2. Нажмите на карандаш ✏️ (Edit this file)

## Что редактировать:

### 1. Добавить/удалить/изменить предметы:
Найдите массив subjects (примерно строки 5-55) и редактируйте:

\\\javascript
let subjects = JSON.parse(localStorage.getItem(STORAGE_KEY)) || [
    {
        id: 1,
        name: "Название предмета",
        teacher: "Имя преподавателя",
        files: []  // массив файлов
    },
    // ... другие предметы
];
\\\

### 2. Добавить файл к предмету:
1. Сначала загрузите файл в папку \/files/\ на GitHub
2. Затем добавьте в массив \iles\ предмета:

\\\javascript
files: [
    {
        name: "Понятное название для студентов",
        desc: "Описание задания",
        filename: "точное_имя_файла.pdf"  // должно совпадать с загруженным
    }
]
\\\

### 3. Структура предмета:
\\\javascript
{
    id: 1,                    // уникальный номер
    name: "Математика",       // название предмета
    teacher: "Иванов И.И.",   // преподаватель
    files: [                  // массив файлов (может быть пустым)
        {
            name: "Задание 1",
            desc: "Решить задачи 1-10",
            filename: "math_task1.pdf"
        }
    ]
}
\\\

## Процесс обновления:
1. Внесите изменения в файл \data.js\
2. Внизу страницы GitHub:
   - Введите описание изменений
   - Нажмите **Commit changes**
3. Подождите 1-2 минуты
4. Обновите сайт: Ctrl+F5
