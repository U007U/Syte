ПРОСТАЯ ИНСТРУКЦИЯ ПО РЕДАКТИРОВАНИЮ
=====================================

1. ОТКРЫТЬ ФАЙЛ ДЛЯ РЕДАКТИРОВАНИЯ:
   https://github.com/U007U/Syte/edit/main/assets/js/data.js

2. НАЙТИ СПИСОК ПРЕДМЕТОВ:
   Прокрутите до строк 10-60, найдите:
   let subjects = [ ... ];

3. ЧТО МОЖНО ДЕЛАТЬ:

   А) ДОБАВИТЬ ПРЕДМЕТ:
      {
          id: 6,                    // следующий номер
          name: "Новый предмет",
          teacher: "Преподаватель",
          files: []                 // пока пусто
      },

   Б) УДАЛИТЬ ПРЕДМЕТ:
      Просто удалите блок { ... }, с этим предметом

   В) ИЗМЕНИТЬ ПРЕДМЕТ:
      Отредактируйте name или teacher

   Г) ДОБАВИТЬ ФАЙЛ:
      1. Загрузите файл в папку /files/ на GitHub
      2. Добавьте в массив files:
         {
             name: "Название для студентов",
             desc: "Описание",
             filename: "имя_файла.pdf"  // точно как загрузили
         }

4. СОХРАНИТЬ:
   - Внизу страницы введите описание изменений
   - Нажмите "Commit changes"

5. ПРОВЕРИТЬ:
   - Подождите 1-2 минуты
   - Обновите сайт: Ctrl+F5

СКРИПТЫ ДЛЯ ПОМОЩИ:
===================
- show-subjects.ps1   - показывает все предметы
- add-subject-template.ps1 - создает шаблон нового предмета
